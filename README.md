# Lejdborg's dotfiles

## Install

    git clone git://github.com/Lejdborg/dotfiles ~/.dotfiles
    cd ~/.dotfiles
    git submodule update --init
    bundle install
    rake install

## Vim

Vim plugins are managed by Vundle. To install, open Vim and run `:PluginInstall`.
